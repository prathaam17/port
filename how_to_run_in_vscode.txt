How to Run the Logistics Management System in VS Code

=========================================
STEP 1: OPEN THE PROJECT IN VS CODE
=========================================
1. Open Visual Studio Code (VS Code).
2. Click on "File" > "Open Folder..." in the top menu.
3. Navigate to your project folder: "c:\Users\prath\Desktop\final port" and click "Select Folder".

=========================================
STEP 2: START THE BACKEND SERVER
=========================================
1. Ensure PostgreSQL is installed and running on your computer.
2. Create a database named `port_inventory` using pgAdmin, psql, or any PostgreSQL administration tool.
3. Open the `backend/.env` file in VS Code and check the PostgreSQL connection settings (username, password, host, port) to make sure they match your server.
4. In VS Code, open a new terminal by clicking on "Terminal" > "New Terminal" from the top menu (or press Ctrl + `).
5. In the terminal window that appears at the bottom, navigate to the backend folder by typing:
   cd backend

6. (First time only) Install all the necessary server packages by running:
   npm install

7. Start the backend server by running:
   npm start

8. You should see a message in the terminal indicating that the server is running (e.g., "Server running on port 5000" and "Database connection has been established successfully.").
   Note: Keep this terminal open!

=========================================
STEP 3: START THE FRONTEND (USER INTERFACE)
=========================================
1. You need a second terminal for the frontend. In the terminal panel at the bottom right, click the "+" icon (or the split-screen icon) to open a new terminal alongside your backend terminal.
2. In this new terminal, navigate to the frontend folder by typing:
   cd frontend

3. (First time only) Install all the necessary frontend packages by running:
   npm install

4. Start the frontend React application by running:
   npm run dev

5. You should see a message showing a local web address, usually something like "http://localhost:5173" or "http://localhost:3000".
   Note: Keep this terminal open as well!

=========================================
STEP 4: ACCESS THE APPLICATION
=========================================
1. Hold the "Ctrl" key and click on the local link provided in your frontend terminal (e.g., http://localhost:5173).
2. Alternatively, open your web browser (Chrome, Edge, etc.) and type that link directly into the address bar and hit Enter.
3. The application will now load in your browser and you can start using it!

=========================================
TROUBLESHOOTING & TIPS
=========================================
- Closing Terminals: If you close the terminal windows in VS Code, the app will stop working. You must leave both the backend and frontend terminals running in the background.
- "npm is not recognized" Error: If you get this error, you need to download and install Node.js from https://nodejs.org/ onto your computer, then restart VS Code.
- Database Errors: If the backend fails to connect to the database, ensure your PostgreSQL server is running locally and that the credentials (username, password, port, host) in your backend/.env file match your local PostgreSQL server configuration.
